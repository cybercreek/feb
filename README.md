Port your favourite applications over to ubuntu!.

We are always working on new updates, so be sure to keep an eye open!.


## Installation

``git clone https://github.com.com/cybercreek/puntu.git``

``chmod +x quickstall.sh``

`` sudo ./quickstall.sh``
